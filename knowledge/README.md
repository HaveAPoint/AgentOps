# Knowledge Repository

这是当前仓库的知识笔记入口。

目标：

- 把一次性对话沉淀成可复用知识
- 让 Go / SQL / 项目实战知识可检索
- 优先记录当前项目里已经实际遇到的问题

目录约定：

- [`index.md`](./index.md)
  仓库总索引
- [`go/`](./go/)
  Go 语言与标准库
- [`database/`](./database/)
  SQL / PostgreSQL / 事务 / 锁 / 数据建模
- [`project/`](./project/)
  AgentOps 项目中的接口、表结构、设计取舍与错误案例

维护原则：

- 优先记录已经出现过的问题
- 优先记录最佳实践、原因和易错点
- 一个主题尽量只写一篇主笔记，相关内容通过索引交叉引用
- 避免纯流水账，避免把临时结论直接写死成长期规则
